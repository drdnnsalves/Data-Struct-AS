# Data-Struct-AS

## Implementação de Biblioteca de Árvore Rubro-Negra

Este repositório contém uma biblioteca em C++ para uma Árvore Rubro-Negra, uma estrutura de dados balanceada que combina as propriedades das árvores de busca binária com restrições adicionais garantindo balanceamento. As Árvores Rubro-Negras são eficientes para operações de inserção, remoção e busca, mantendo um balanceamento automático que assegura tempos de acesso previsíveis.

### Estrutura do Repositório

- `functionsRedBlack.cpp`: Implementação das funções principais da Árvore Rubro-Negra.
- `functionsRedBlack.h`: Arquivo de cabeçalho contendo as definições das funções da Árvore Rubro-Negra.
- `redBlackTree.cpp`: Arquivo principal.

### Funcionalidades Implementadas

- **Inserção**: Permite a inserção de um novo nó na Árvore Rubro-Negra mantendo suas propriedades.
- **Remoção**: Suporta a remoção de um nó existente enquanto preserva as propriedades da Árvore Rubro-Negra.
- **Busca**: Buscar um nó com valor específico na árvore.
- **Percurso em Ordem Simétrica (inorder)**: Realiza um percurso em ordem simétrica na árvore, mostrando os valores dos nós em ordem crescente.
- **Validação**: Verifica se uma árvore é de fato uma Árvore Rubro-Negra.
- **Encontrar Nó Máximo e Mínimo**: Encontra o nó máximo e mínimo na árvore.
- **Cálculo de Altura**: Calcula a altura da Árvore Rubro-Negra.

### Testes

Testes foram realizados para validar a funcionalidade das operações implementadas. Os testes cobrem cenários que incluem inserção, remoção, busca e várias configurações da estrutura da árvore.
